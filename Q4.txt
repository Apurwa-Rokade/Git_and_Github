GitHub is a web-based platform that provides hosting for Git repositories and offers a range of 
collaboration and version control tools. It facilitates collaboration among developers, enabling them 
to work on projects together, manage code, track changes, and contribute to open-source projects.

Features of GitHub:
1)Repository Hosting: Allows users to host Git repositories remotely, providing a central place for 
project files and version control.
2)Collaboration Tools: Offers features like issues, pull requests, and project boards that streamline 
communication and collaboration among team members.
3)Version Control: Provides powerful version control capabilities through Git, enabling tracking of 
changes, branching, merging, and reviewing code history.
4)Community & Open Source: Fosters a large community of developers, making it a hub for open-source 
projects, allowing contributions from anyone worldwide.

Facilitating Collaboration:
Pull Requests: Developers propose changes, and collaborators review, comment, and merge those changes 
into the main branch.

Issues: Used to report bugs, request features, or discuss ideas, fostering discussion and tracking tasks.

Forks: Users can create copies of repositories, allowing them to experiment with changes without
affecting the original codebase.
